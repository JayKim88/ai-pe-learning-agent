**Q: "How do you think about human-AI interaction UX?"**

   - Users can see exactly why code failed or passed

2. **Control:** Let users override AI decisions
   - My system suggests assignments but users can reject/modify
   - Humans always have final say

3. **Progressive Disclosure:** Don't overwhelm with AI complexity
   - Simple interface: submit code ‚Üí get feedback
